# Getting Started

Install modules:

```bash
> npm i
```

Install Husky

```bash
> npx husky install
```

Add dev.env prod.env
```bash
APP_EXTERNAL_PORT=3000
APP_INTERNAL_PORT=3000
APP_IMAGE_NAME=
APP_CONTAINER_NAME=
```

Run the development server:

```bash
> npm run dev
```

Or run docker-compose
```bash
> npm run docker-dev
```
